What is SPARQL?

SPARQL (SPARQL Protocol and RDF Query Language, pronounced “sparkle”) is a query language used to retrieve and manipulate linked data stored in an RDF format. In this format, data are stored as triples. Triples are formed by a suject, predicate, and object.


Example:

The best way to get started is to write simple a simple query:

SELECT ?subject ?predicate ?object
WHERE {
    ?subject ?predicate ?object .
}

This query will return all the triples in the dataset.

Let's break down this query further:

?:

Any word following the question mark is a variable.

SELECT:

The variables following SELECT will be the ones displayed in a table. There are other keywords which could be used here and will return something else like CONSTRUCT, ASK, and DESCRIBE.

WHERE:

The WHERE clause is always mandatory and specifies the data to pull out. Other temporary variables may be used here.

Note: This might result is a lot triples, we can use modifiers such as LIMIT to refine our search. This topic will be explored in greater detail in the next chapter. The following will display the first 10 results.

SELECT ?subject ?predicate ?object
WHERE {
    ?subject ?predicate ?object .
}
LIMIT 10

Prefixes and URIs

The data are stored in triples, which have subject, predicate, object format. Each part of the triple is identified by Uniform Resource Identifier (URI). For example, being a woman in the CWRC Ontology is represented by the following URI:

<http://sparql.cwrc.ca/ontologies/cwrc/woman>

To save space and for clarity, the URIs in queries are often shortened using a prefix defined at the top of the query which stands in for most of the URI except the changed endings.

The parts of a simple SPARQL query

PREFIX cwrc: <http://sparql.cwrc.ca/ontologies/cwrc#>

SELECT ?person
WHERE {
   ?person cwrc:hasGender cwrc:woman .
   ?person cwrc:hasOccupation ?occupation .
   ?occupation a cwrc:teacher .
}

This query searches of all the triples in the dataset that satisfy the following requirements: the person is a woman and a teacher.

In plain English, it could be translated as follows:

There is a person who has a gender of woman

That same person has an occupation

That occupation is teacher

Notes on Style:

Every triple in a query should end in a period.

If a variable use in one line is repeated in the next line it can be omitted. Example:

PREFIX cwrc: <http://sparql.cwrc.ca/ontologies/cwrc#>

SELECT ?person ?occupation
WHERE {
   ?person cwrc:hasGender cwrc:woman .
        cwrc:hasOccupation ?occupation .
}
